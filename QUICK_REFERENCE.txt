│                                                                              │

│  故障排查:        bash /root/openclaw-tool/openclaw-troubleshoot.sh         │

│    Tailscale:     bash /root/openclaw-tool/fix-tailscale-routing.sh         │

┌─ 📊 服务管理 ────────────────────────────────────────────────────────────────┐
│                                                                              │
│  查看状态:        openclaw status                                           │
│  查看日志:        openclaw logs --follow                                    │
│  测试连接:        openclaw gateway probe                                    │
│                                                                              │
│  启动服务:        systemctl start openclaw-gateway.service                  │
│  停止服务:        systemctl stop openclaw-gateway.service                   │
│  重启服务:        systemctl restart openclaw-gateway.service                │
│  查看日志:        journalctl -u openclaw-gateway.service -f                 │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

│  查看模型:        openclaw models status                                    │
│                                                                              │
│  配置文件:        /root/.openclaw/openclaw.json                             │
│  Nginx 配置:      /etc/nginx/sites-available/openclaw-gateway.conf          │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ 🐛 常见问题 ────────────────────────────────────────────────────────────────┐
│                                                                              │
│  Tailscale 路由错误:                                                         │
│    → bash /root/openclaw-tool/fix-tailscale-routing.sh                      │
│                                                                              │
│  Origin 不允许:                                                              │
│    → bash /root/openclaw-tool/fix-gateway-origin.sh                         │
│                                                                              │
│  认证失败锁定:                                                               │
│    → bash /root/openclaw-tool/reset-gateway-token.sh                        │
│                                                                              │
│  Gateway 无法连接:                                                           │
│    → systemctl restart openclaw-gateway.service                             │
│    → openclaw gateway probe                                                 │
│                                                                              │
│  API 密钥错误:                                                               │
│    → openclaw config set env.ANTHROPIC_API_KEY "your-key"                   │
│    → systemctl restart openclaw-gateway.service                             │
│                                                                              │
│  Nginx 502 错误:                                                             │
│    → systemctl status openclaw-gateway.service                              │
│    → tail -f /var/log/nginx/openclaw-error.log                              │
│                                                                              │
│  Telegram 不响应:                                                            │
│    → openclaw pairing list                                                  │
│    → openclaw pairing approve <request-id>                                  │
│    → openclaw config set channels.telegram.allowFrom '["USER_ID"]'          │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ 🔐 安全配置 ────────────────────────────────────────────────────────────────┐
│                                                                              │
│  Gateway Token:   openclaw config get gateway.auth.token                    │
│  更新 Token:      openclaw config set gateway.auth.token "$(openssl rand -hex 24)"
│                                                                              │
│  Telegram 白名单:                                                            │
│    1. 发送消息给 Bot                                                         │
│    2. openclaw logs --follow  (获取 USER_ID)                                │
│    3. openclaw config set channels.telegram.allowFrom '["USER_ID"]'         │
